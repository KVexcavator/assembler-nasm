### 1. Установите инструменты разработки:
```bash
sudo apt update
sudo apt install nasm gcc gdb
sudo apt install gcc-multilib
```

### 2. Добавить  код на ассемблере


### 3. Скомпилировать код
Скомпилируйте файл с помощью `nasm`, а затем используйте `gcc` для компоновки.

1. Ассемблировать код:
   ```bash
   nasm -f elf32 hello.asm -o hello.o
   ```

2. Компоновка кода `gcc`:
   ```bash
   gcc -m32 hello.o -o hello
   ```

Здесь `-f elf32` указывает на формат вывода для 32-битного Linux, а `-m32` — это флаг для компиляции под 32-битную архитектуру. Если  64-битная система, обязательно добавьте этот флаг.

### 4. Запуск
```bash
./hello
```
Ввывод:
```
Hello, world!
```
### 5. Отладка программы (опционально)
```bash
gdb ./hello
```
