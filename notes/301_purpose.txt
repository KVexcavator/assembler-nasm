Назначение регистров
Общего назначения
а(accumulator ) - хранение промежуточных данных
b(base) -  хранение базового адреса некого объекта в памяти
c(count) - счетчик для повторяющихся действий
d(data) - хранение промежутояных данных
Цепочных инструкций
si(source index) - индекс источника, содержит текущий адрес в цепочке источника
di(destination index) - индекс назначения, содержит текущий адрес в цепочке получателя
Для работы со стеком
sp(stack pointer ) - указатель вершины стека в текущем сегменте стека
bp(base pointer) - адрес начала текущего сегмента стека
Состояния и управления
ip(indtruction pointer) - указатель команд, содержит адрес следующей инструкции
Флаги
eflags/flags - состояния, управления,системные
